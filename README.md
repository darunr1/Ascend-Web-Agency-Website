# Ascend Web Agency Website

A modern, unique website for Ascend Web Agency featuring a stunning teal-to-lime green gradient color scheme inspired by the agency's logo.

## Features

- **Modern Design**: Unique, eye-catching design with smooth animations and gradients
- **Responsive**: Fully responsive layout that works on all devices
- **Interactive Elements**: Smooth scrolling, hover effects, and animated components
- **Portfolio Showcase**: Features the Lang Son Traders project with interactive preview
- **Contact Integration**: Direct links to Tally contact form and email

## Color Scheme

The website uses the teal-to-lime green gradient from the Ascend Web Agency logo:
- **Teal**: `#14b8a6`
- **Lime Green**: `#84cc16`
- **Dark Background**: `#0f172a`

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling and animations
- `script.js` - Interactive functionality and animations

## How to Use

1. Open `index.html` in a web browser
2. The website is fully functional and ready to use
3. All links are configured:
   - Contact form: Links to Tally form (https://tally.so/r/2EXdoA)
   - Email: ascendwebagency1@gmail.com
   - Portfolio: Links to Lang Son Traders website

## Sections

- **Hero**: Eye-catching introduction with animated gradient orbs
- **Services**: Six service cards showcasing agency capabilities
- **Portfolio**: Interactive showcase of Lang Son Traders project
- **Contact**: Contact information and call-to-action buttons
- **Footer**: Additional links and information

## Browser Support

Works on all modern browsers including:
- Chrome
- Firefox
- Safari
- Edge

## Customization

The website is easy to customize:
- Colors are defined in CSS variables at the top of `styles.css`
- Content can be modified directly in `index.html`
- Animations and interactions are in `script.js`
